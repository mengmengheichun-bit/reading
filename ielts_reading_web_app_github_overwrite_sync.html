Tên bài / tên file

Ghi tên bài ở đây để sau này nhìn dropdown là biết bài này thuộc file nào.

1) Upload JSON

Cách chắc nhất: upload JSON bài lẻ hoặc JSON workspace đã export từ máy khác/GitHub. App sẽ giữ bài trong trình duyệt.

2) Import JSON từ GitHub/link

Dán link file JSON trên GitHub Pages hoặc raw GitHub. Nếu để file tên library.json cùng thư mục với index.html, web sẽ tự thử load khi mở.

3) Upload DOCX / Paste text

DOCX chỉ import raw text và tự đoán nhóm câu hỏi cơ bản. Muốn chấm điểm thì paste answer key dạng 1=A.

DOCX parser dùng Mammoth CDN nếu có internet. Nếu không được, copy text từ Word rồi paste vào ô bên dưới.
